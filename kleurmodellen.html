Kleurmodellen

Computers en Printers gebruiken veel verschillende kleurmodellen om kleuren weer te geven. De twee die het vaakst voorkomen zijn:

  • RGB (Rood, Groen en Blauw): deze wordt het vaakst gebruikt voor weergave van een scherm.
  • CMY-K (Cyaan, Magenta, Geel, Zwart): deze wordt vooral gebruikt voor het printen.

Ook zijn pixels en resolutie erg belangrijk.


RGB Kleurmodel

Op een scherm gebruikt elke pixels drie subpixels. Dat zijn rood, groen en blauw. Door de ene kleur feller te maken dan de ander kunnen alle mogelijke kleurencombinaties worden weergegeven. Deze felheid wordt meestal gemeten op een schaal van 0 tot 255. Een aantal voorbeelden:

  • Zwart: (0, 0, 0)
  • Wit: (255, 255, 255)
  • Rood: (255, 0, 0)
  • Groen: (0, 255, 0)
  • Blauw: (0, 0, 255)
  • Geel: (255, 255, 0)

CMY-K Kleurmodel

CMY-K werkt met subtractieve kleurmenging. Dit betekent dat kleuren worden gemaakt met licht van een wit oppervlak af te trekken met inkt. Zwart (K) wordt toegevoegd om verschillende kleuren te maken. Een aantal voorbeelden:

  • Wit: (0, 0, 0, 0)
  • Zwart: (0, 0, 0, 100)
  • Rood: (0, 100, 100, 0)
  • Groen: (100, 0, 100, 0)
  • Blauw: (100, 100, 0, 0)
  • Geel: (0, 0, 100, 0)

Pixels

Pixels zijn de kleinste eenheid van een digitaal beeld. Een scherm is opgebouwd uit duizenden tot zelfs wel miljoenen pixels. Al deze pixels kunnen elke een kleur weergeven.

De kwaliteit van het beeld word bepaald door het aantal pixels (de resolutie).


Resolutie

Resolutie is het aantal pixels in een scherm of een afbeelding. Het wordt vaak weergegeven door breedte x hoogte. Een aantal voorbeelden:

  • HD: 1280 x 720 pixels
  • Full HD: 1920 x 1080 pixels
  • 4K Ultra HD: 3840 x 2160 pixels
  • 8K Ultra HD: 7680 x 4320 pixels
